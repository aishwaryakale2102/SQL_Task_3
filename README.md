# SQL for Data Analysis

This project analyzes an ecommerce dataset using SQL in MySQL Workbench.

## Tools Used
- MySQL Workbench
- CSV Dataset

## SQL Concepts Used
- SELECT
- WHERE
- GROUP BY
- ORDER BY
- Aggregate Functions
- Views

## Key Insights
- Identified top-selling products
- Found country-wise sales
- Calculated average product prices
- Created a view for top products
